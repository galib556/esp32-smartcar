# 📷 Media

This folder contains build photos and demo videos for the ESP32 SmartCar project.

## 📁 Structure

```
media/
├── photos/    ← Add your build & demo photos here (.jpg, .png)
└── videos/    ← Add your demo video clips here (.mp4, .gif)
```

## 📸 Suggested Photos to Add

- Top-down view of the assembled car
- Close-up of the ESP32 and L298N wiring
- Close-up of the motor connections
- Screenshot of the web UI on a phone
- Car in action (motion blur shot)

## 🎥 Suggested Videos to Add

- 60–90 second demo: connecting to WiFi, opening UI, driving the car
- Short clip showing turning/reversing
- Screen recording of the web UI while driving

> **Tip:** A short GIF demo embedded in the README makes a huge impression on recruiters. Record a clip and use [ezgif.com](https://ezgif.com/video-to-gif) to convert it, then add it to README.md like:
> ```markdown
> ![Demo GIF](media/videos/demo.gif)
> ```
